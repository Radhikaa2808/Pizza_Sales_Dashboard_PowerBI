 **Pizza Sales Insights Dashboard - Power BI**
 
**Project Overview**
This project analyzes pizza sales data to uncover key business insights using Power BI.
The goal is to evaluate sales performance, identify top and bottom products, and understand customer ordering patterns.

🔹 Category Insights
Sales distribution by pizza category
Sales distribution by pizza size

**Key Insights**
Classic category contributes the highest sales and orders
Large size pizzas generate the most revenue
Peak orders occur on weekends, especially evenings
Certain pizzas consistently underperform and may need review
